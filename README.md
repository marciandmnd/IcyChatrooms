# Icy Chatrooms

Chatroom application built using Node and Socket.io.

## Getting Started

`git clone git@github.com:marciandmnd/chatrooms.git`
`cd chatrooms`
`npm install`
`node server.js`

## Built With

* [Node.js](https://nodejs.org/en/)
* [socket.io](https://socket.io/)
